# 📦 Panzer Robotics IoT 
Website monitoring Internet Of Things Panzer Robotics

## ✨ Features  
- Login, Register, Forgot Password with Fortify


## ⚙️ Installation & 🚀 Usage 
- Clone Project
```
git clone https://github.com/ASNProject/panzer-web-iot.git
```
- Buka Project
```
cd project_name
```
- Install composer
```
composer update
```
- Run Project
```
php artisan serve
```
